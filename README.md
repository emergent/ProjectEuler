# Codes of Project Euler solution

## Usage

### Haskell

```
$ runghc problem001.hs
```

### Clojure

Use clojure >= 1.9.0.

```
$ clojure problem001.clj
```

### Python

Use `python3`.

```
$ python3 problem001.py
```

### C/C++

```
$ make
$ ./problem001
```

### Ruby

```
$ ruby problem001.rb
```

### Scheme

Use `Gauche`.

```
$ gosh problem001.scm
```
