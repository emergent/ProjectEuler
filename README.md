# Codes of Project Euler solution

## Usage

### Haskell

```
$ runghc problem001.hs
```

### Clojure

Install `boot-clj` before executing program.

```
$ chmod a+x problem001.clj
$ ./problem001.clj
```

### Python

Use `python3`.

```
$ python3 problem001.py
```

### C++

```
$ make
$ ./problem001
```
